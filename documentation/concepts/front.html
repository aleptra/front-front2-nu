Framework Overview

Front's runtime executes FTML markup, manages modules and plugins, and handles reactivity and event propagation efficiently.

Front vs Tradional Frameworks/Libraries

Aspect Front (FTML) React Vue Angular
Approach Markup-first, declarative Component + JSX Component + template syntax Component + TypeScript, MVVM
Setup No build tools required Requires bundler (Webpack/CRA) Optional bundler, can use CDN CLI setup, TypeScript configuration
Learning Curve Easy if you know HTML Moderate, JSX and props/state concepts Moderate, template syntax and reactivity Steep, TypeScript, DI, RxJS
Development Style Express behavior directly in markup Split between JSX, state, and logic Split between templates, scripts, and reactive data MVVM, TypeScript classes, services
Prototyping Speed Very fast Medium Medium Slower due to setup and boilerplate
Best For Data-driven apps, rapid development, collaboration Dynamic SPAs Dynamic SPAs and progressive apps Large-scale enterprise apps
