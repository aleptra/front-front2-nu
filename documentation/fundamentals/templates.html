Templates define reusable regions or content blocks that act as blueprints. They allow replacing, removing or adding FTML attributes and content within the layout foundation provided by the Start Page, separating layout from content and enabling reuse across your application.

Example:

template1.html <template>  <header></header>  <aside></aside>  <main></main>  <aside></aside>  <footer></footer> </template> This defines a reusable layout with a header, two sidebars, a main section, and a footer.
Note: If an attribute is included without a value, it will be removed from the final output.
