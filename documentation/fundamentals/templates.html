Templates define reusable regions or content blocks that act as blueprints. They allow replacing, removing or adding FTML attributes and content within the layout foundation provided by the Start Page, separating layout from content and enabling reuse across your application.

Usage

Once you have defined a template file (e.g. template1.html), you can include it in another FTML document using the >template src="..."> attribute:

<template src="template1.html"></template> This will render the structure defined in template1.html and allow replacing, adding, or removing elements as needed.

Example:

template1.html <!-- Defines a reusable layout --> <template>  <header></header>  <aside></aside>  <main include></main>  <aside></aside>  <footer></footer> </template>
Note: If an FTML attribute is without a value, it will be removed from the final output.
