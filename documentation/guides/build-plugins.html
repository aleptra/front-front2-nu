This example shows how to build a simple plugin in Front using the `pluginname--load` convention. The plugin will log a message and modify a page element.

Example: Hello Plugin

hello.js 'use strict' app.plugin.hello = { __autoload: function (options) { app.log.info()("Hello plugin loaded!") } load: function (options) { var element = options.element if (element) { element.innerHTML = "Hello from Front plugin!" element.style.backgroundColor = "#f0f0f0" } } }

Usage:

page1.html <html>  <head>   <script src="https://cdn.front.nu/nightly/front.js"></script>  </head>  <body>   <!-- Set the page title -->   <h1>Hello world!</h1>  </body> </html>
Note: The `pluginname--load` method is the entry point of your plugin. You can define additional methods using the same `pluginname--method` convention.
