Front uses standard JavaScript (ES5) and extends it with simple helpers for common DOM tasks.

You should understand basic concepts like var, function, objects, arrays, and events.

If you're new to JavaScript, check MDN Web Docs ↗️.

Example: Hello Plugin

This example shows how to build a simple plugin in Front using the `pluginname--load` convention. The plugin will log a message and modify a page element.

hello.js 'use strict' app.plugin.hello = { // Automatically triggered when the plugin file is loaded. __autoload: function (options) { app.log.info()('Hello plugin loaded!') } load: function (options) { var element = options.element if (element) { element.innerHTML = 'Hello from Front plugin!' element.style.backgroundColor = '#f0f0f0' } } }

Usage:

page1.html <html> <head> <!-- Load Front with hello plugin --> <script src="https://cdn.front.nu/nightly/front.js" plugin="hello" conf="debug:true"> </script> </head> <body> <!-- Load function from plugin --> <h1 hello--load></h1> </body> </html>
Note: The `pluginname--load` method is the entry point of your plugin. You can define additional methods using the same `pluginname--method` convention.
