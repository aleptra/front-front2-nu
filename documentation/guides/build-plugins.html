This example shows how to build a simple plugin in Front using the `pluginname--load` convention. The plugin will log a message and modify a page element.

Example: Hello Plugin

hello.js 'use strict' app.plugin.hello = { "hello--load": function (options) { var element = options.element if (element) { element.innerHTML = "Hello from Front plugin!" element.style.backgroundColor = "#f0f0f0" } console.log("Hello plugin loaded!") } }

Usage in your page:

<script src="hello.js"></script> <div id="my-element"></div> <script> app.plugin.hello["hello--load"]({ element: document.getElementById("my-element") }) </script>
Note: The `pluginname--load` method is the entry point of your plugin. You can define additional methods using the same `pluginname--method` convention.
