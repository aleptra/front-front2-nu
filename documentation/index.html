2. Get started

Welcome! Here you'll find step-by-step guides for setting up your development environment, installing dependencies, and creating your first application using the Front CLI tool. Whether you're a new or experienced developer, follow the instructions below to get started on macOS, Windows, or Linux/Unix.

Automatic Installation

Set up development environment automatically using GIT and Front CLI tool.

➊ Install GIT on your system:
  • macOS
  • Windows
  • Linux/Unix
➋ Clone project:
git clone https://github.com/aleptra/front.git
➌ Create an application:
make app:create
