2. Introduction

Front is a powerful library for building interactive and dynamic applications using declarative HTML-like syntax. It extends standard HTML with FTML (Front Text Markup Language), allowing developers to control content, style, layout, interactivity, and behavior through attributes, and to extend functionality with modules and plugins.


Attributes in FTML are organized into three layers:

  • Core Attributes - fundamental controls for content, style, and basic behavior.
  • Module Attributes - feature-specific extensions that provide domain-level functionality, such as responsive design and layout.
  • Plugin Attributes - external or optional attributes that enable custom behaviors or third-party features.
