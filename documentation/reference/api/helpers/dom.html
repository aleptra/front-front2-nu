Front provides a set of DOM helper methods that simplify element selection, creation, and manipulation. These helpers are designed to work consistently across pages and plugins, following ES5 conventions. You can use these instead of native document.getElementById, querySelector, etc.

Getting Elements

Use dom.get(selector) to retrieve the first matching element, and dom.find(selector) to get all matching elements.

Example: var main = dom.get('main'); var buttons = dom.find('.btn');

Creating and Removing Elements

Use dom.create(tagName, attributes) to create new elements and dom.remove(element) to remove them.

Example: var div = dom.create('div', { id: 'myDiv', class: 'card' }); dom.get('main').appendChild(div); dom.remove(div); // removes the element

Manipulating Attributes and Content

Use the helpers to get/set attributes or text content.

Example: var header = dom.get('header'); dom.attr(header, 'data-role', 'main'); // set attribute var role = dom.attr(header, 'data-role'); // get attribute dom.text(header, 'Welcome!'); // set text content var title = dom.text(header); // get text content
